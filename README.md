# BashaBos Kids Fashion Store
### Hopscotch-Inspired E-Commerce | React 18 + Vite | Razorpay

## 🚀 Quick Start
```bash
npm install
npm run dev
```

## 💳 Razorpay Setup
In `src/pages/Checkout.jsx`, replace:
```
key: 'rzp_test_YOUR_KEY_HERE'
```
Get your key from: https://dashboard.razorpay.com
